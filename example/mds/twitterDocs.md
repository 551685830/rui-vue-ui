## 使用

```html
<mj-time-line :option="option"> </mj-time-line>
```

## API

| 参数   | 说明   | 类型   | 默认值                                                                                  | 必填  | 返回值 |
| ------ | ------ | ------ | --------------------------------------------------------------------------------------- | ----- | ------ |
| option | 配置项 | Object | {screenName: '用户名',height: '卡片高度', follow: true, imgUrl: '加载失败时候显示图片'} | false | -      |
